# Assets

Esta pasta centraliza todos os recursos visuais, sonoros e tipográficos utilizados no jogo.


## Organização

- `imagens/`: sprites, fundos, ícones e elementos visuais do jogo.
- `sons/`: efeitos sonoros e músicas utilizadas durante o jogo.
- `fontes/`: fontes tipográficas utilizadas na interface.


## Origem dos recursos

- Imagens (sprites e elementos visuais): geradas com auxílio de inteligência artificial (ChatGPT) e adaptadas para o projeto.
- Sons e músicas:
  - IA (ElevenLabs): https://elevenlabs.io/app/home
  - Pixabay: https://pixabay.com/pt/
  - Itch.io (trilhas musicais): https://hzsmith.itch.io/free-medieval-epic-loops
- Fontes tipográficas: Google Fonts (- https://fonts.google.com/specimen/Press+Start+2P)
